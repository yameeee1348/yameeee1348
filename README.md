## Core Projects
참여한 핵심 레포지토리 및 주요 역할이다.

### 1. Ondevice_UVM
* **Link**: [Ondevice_UVM](https://github.com/yameeee1348/Ondevice_UVM)
* **Description**: 하드웨어 IP 기능 검증을 위한 UVM 환경 구축 및 모니터 구현 프로젝트이다.

### 2. Ondevice_RTL
* **Link**: [Ondevice_RTL](https://github.com/yameeee1348/Ondevice_RTL)
* **Description**: 하드웨어 제약 환경을 고려한 인공지능 모델 구조 재설계 및 RTL 레벨 디지털 회로 설계 프로젝트이다.

### 3. M.A.T.S.
* **Link**: [M.A.T.S.](https://github.com/kccistcs1st/M.A.T.S.)
* **Description**: FPGA를 활용한 AI 기반 포트홀 탐지 및 경고 시스템 캡스톤 디자인 프로젝트이다.

### 4. UI_system
* **Link**: [UI_system](https://github.com/kccistcs1st/UI_system)
* **Description**: 차량용 통신 및 탐지 모델과 연동되는 UI 시스템 구축 프로젝트이다.
